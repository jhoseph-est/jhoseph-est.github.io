El operador derivada. Definición. Propiedad fundamental.

El operador derivada se denota por
$D = \frac{d}{dx}$ y representa la acción de diferenciar una función respecto a una variable, normalmente $x$.

Si $y = f(x)$, entonces: $$ D(f) = \frac{df}{dx} = f^{\prime} (x) $$

Es decir, el operador derivada transforma una función en otra que mide su tasa de cambio instantánea.

Propiedad fundamental

El operador derivada es lineal, lo que significa que cumple: $$ D(af + bg) = aD(f) + bD(g) $$ para cualesquiera constantes $a , b$ y funciones $f, g$ .

Además, cumple las reglas de derivación:

  • Producto: $D(fg) = f^{\prime} g + f g^{\prime}$
  • Cociente: $D\left(\frac{f}{g}\right) = \frac{f^{\prime} g - f g^{\prime}}{g^2}$
  • Composición: $D(f(g(x))) = f^{\prime} (g(x)) \cdot g^{\prime} (x)$

La inversa del operador derivada. Interpretación: La antiderivada, la primitiva.

Definición

la antiderivada o función primitiva de $f(x)$ se denota por $Antd (f(x))$ y su valor esta definido por:

$Antd (f(x)) = F(x) + k$ tal que $F'(x) = f(x)$ en cierto intervalo ${\mathbb{I}}$

antiderivadas

$$ Antd (a f(x)) = a \cdot Antd (f(x)) ; a = cte $$ $$ Antd (f(x) \pm h(x)) = Antd (f(x)) \pm Antd(h(x)) $$ $$ Antd \left( \frac{f(x)}{F(x)} \right) = \ln \left| F(x) \right| $$

Integración de funciones racionales: Descomposición en fracciones parciales.

puedes transformas una fracción racional impropia a propia de la siguiente forma:

$$ \int \frac{P(x)}{Q(x)} dx = \int \left[ C(x) + \frac{R(x)}{Q(x)} \right] dx = \int C(x) dx + \int \frac{R(x)}{Q(x)} dx $$

CASO 1

$$ \frac{Q(x)}{(a_{1}x + b_{1})(a_{2}x + b_{2})(a_{3}x + b_{3}) \dots (a_{n}x + b_{n})} = \frac{A_{1}}{(a_{1}x + b_{1})} + \frac{A_{2}}{(a_{2}x + b_{2})} + \frac{A_{3}}{(a_{3}x + b_{3})} + \dots + \frac{A_{n}}{(a_{n}x + b_{n})} $$

para hallar el valor de $A_k$ se usa el método algebraico

$$ \frac{A_{k}}{a_{k}x + b_{k}} = \frac{Q(x)}{(a_{1}x + b_{1})(a_{2}x + b_{2})(a_{3}x + b_{3}) \dots (a_{n}x + b_{n})} \Bigg| {x = -\frac{b_{k}}{a_{k}}} $$

CASO 2

$$ \frac{Q(x)}{(x + a_{1})^{p}(x + a_{2})^{q} \dots (x + a_{n})^{r}} = \frac{A_{1}}{(x + a_{1})} + \frac{A_{2}}{(x + a_{1})^{2}} + \dots + \frac{A_{p}}{(x + a_{1})^{p}} + \frac{B_{1}}{(x + a_{2})} + \dots + \frac{B_{q}}{(x + a_{2})^{q}} + \frac{C_{1}}{(x + a_{3})} + \dots + \frac{C_{r}}{(x + a_{3})^{r}} $$

CASO 3

$$ \frac{P(x)}{(a_{1}x^{2} + b_{1}x + c_{1})(a_{2}x^{2} + b_{2}x + c_{2}) \dots (a_{n}x^{2} + b_{n}x + c_{n})} = \frac{A_{1}x + B_{1}}{a_{1}x^{2} + b_{1}x + c_{1}} + \frac{A_{2}x + B_{2}}{a_{2}x^{2} + b_{2}x + c_{2}} + \dots + \frac{A_{n}x + B_{n}}{a_{n}x^{2} + b_{n}x + c_{n}} $$

$$ \Delta_{1} < 0 ; , \Delta_{2} < 0 ; , \dots ; , \Delta_{n} = b_{n}^{2} - 4a_{n}c_{n} < 0 $$

CASO 4

$$ \frac{P(x)}{(a_{1}x^{2} + b_{1}x + c_{1})^{p}(a_{2}x^{2} + b_{2}x + c_{2})^{q}} = \frac{A_{1}x + B_{1}}{a_{1}x^{2} + b_{1}x + c_{1}} + \frac{A_{2}x + B_{2}}{(a_{1}x^{2} + b_{1}x + c_{1})^{2}} + \dots + \frac{A_{p}x + B_{p}}{(a_{1}x^{2} + b_{1}x + c_{1})^{p}} + \frac{C_{1}x + D_{1}}{a_{2}x^{2} + b_{2}x + c_{2}} + \dots + \frac{C_{q}x + D_{q}}{(a_{2}x^{2} + b_{2}x + c_{2})^{q}} $$

Fórmulas de reducción o de recurrencia.

Una fórmula de reducción o fórmula de recurrencia es una relación matemática que permite expresar una integral con un parámetro “n” (por ejemplo, un exponente) en función de otra integral del mismo tipo pero con un parámetro menor, como $n-1$ o $n-2$.

En otras palabras:

Es una regla que reduce una integral complicada a otra más simple del mismo tipo, que se puede resolver de manera repetida o recursiva.

Objetivo principal

El objetivo es simplificar integrales que dependen de un exponente o parámetro aplicando un proceso sistemático, paso a paso, hasta llegar a una integral elemental (fácil de resolver).

Idea general

Supón una integral del tipo:

$$ I_{n} = \int f(x , n) , dx $$

Si se encuentra una relación del tipo:

$$ I_{n} = a(n) , I_{n-1} + b(n) , g(x) $$

entonces se dice que esa relación es una fórmula de reducción.
Permite obtener $I_{n}$​ conociendo $I_{n-1}$​.
Este proceso se repite hasta llegar a una integral $I_{0}$​ o $I_{1}$​, que se puede calcular directamente.

$$ \int M(x, \sqrt{ax^{2} + bx + c}) $$
